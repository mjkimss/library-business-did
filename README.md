# Library Openings and Local Business Formation

## What this project is (in plain English)

This summer, I built a complete research project from scratch to answer a
question: when a new public library opens in a neighborhood, does it help
new businesses open nearby too?

To answer this, I:
- Pulled real data from the City of Chicago on library openings and new
  business registrations
- Manually researched and verified the exact opening dates of 7 Chicago
  library branches
- Built a statistical model (in three different programming languages —
  Python, Stata, and R, to double- and triple-check my own work) to measure
  whether business activity changed after each library opened
- Wrote up the findings in a full research memo

**What I found:** no clear evidence that library openings caused a
measurable increase in nearby new businesses — a real, honest result, even
though it wasn't the "exciting" finding I might have hoped for. Getting a
careful, truthful answer (rather than a flashy one) is the actual point of
research like this.

This project was done independently, as preparation for applying to
research assistant positions in economics.

---

## Technical summary

A difference-in-differences analysis of whether new/renovated Chicago
Public Library branches affect local new business registration rates.
